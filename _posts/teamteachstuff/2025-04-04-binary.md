<div style="background-color: #f0f8ff; padding: 1.5rem; border-left: 6px solid #007acc; border-radius: 0.5rem; font-size: 1.25rem; line-height: 2;">
<b>Binary Search</b> is a powerful algorithm used to quickly find an element in a <b>sorted</b> list by repeatedly dividing the search interval in half.

Binary search is **much faster** than linear search for large datasets. Instead of checking one-by-one from start to end, it jumps right to the middle. Thatâ€™s why itâ€™s often referred to as a **"divide and conquer"** algorithm.

## ğŸ Binary Search in Python

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = (left + right) // 2
        print(f"Checking index {mid}: {arr[mid]}")

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1  # Not found

numbers = [1, 3, 5, 7, 9, 11, 13]
print(binary_search(numbers, 7))  # Output: 3
```
